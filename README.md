# GameHub

GameHub — это платформа для поиска, просмотра и оценки видеоигр.  
Проект включает **бэкенд на FastAPI** и **фронтенд на Vue.js**, с поддержкой отзывов, рейтингов и загрузки обложек игр.

## Функциональность

На данный момент GameHub реализует работу с данными без полноценной системы авторизации. Все функции построены на основе CRUD (Create, Read, Update, Delete) и работы с IP адресом пользователя для идентификации авторов отзывов.  

### Игры
- **Просмотр списка игр**: постраничная навигация (pagination) через API.
- **Детальная информация об игре**:
  - Название
  - Год выпуска
  - Жанры
  - Платформы
  - Средний рейтинг
  - Количество отзывов
  - Обложка игры
- **Создание игры** через API.
- **Редактирование игры** через API.
- **Удаление игры** через API.
- **Топ игр** по рейтингу.
- **Недавние игры** — последние добавленные в базу.
- Получение списков **жанров и платформ** для фильтров.

### Отзывы
- **Добавление отзывов на игры**:
  - Рейтинг от 1 до 10
  - Текст отзыва
- **Редактирование и удаление** своих отзывов (идентификация по IP).
- **Просмотр отзывов** с постраничной навигацией.
- **Последние отзывы** с информацией об игре.
- Автоматический **пересчет среднего рейтинга игры** при добавлении, редактировании или удалении отзывов.
- Ограничение: **один отзыв на игру с одного IP**.

### Обложки
- **Загрузка и обновление обложек** для игр.
- Поддерживаемые форматы: **JPEG, PNG**.
- Загруженная обложка доступна по ссылке в данных игры.

### Пользователи и IP
- Временная система идентификации через **IP адрес**.
- Все действия над отзывами проверяются по IP: редактировать и удалять можно только свои отзывы.

### Администрирование
- Все операции доступны через API без авторизации.
- Настроен **CRUD для игр и отзывов**.
- Проект поддерживает **автоматическое тестирование через CI/CD**.

## Технологии

### Backend
- **Python 3.12**
- **FastAPI** — веб-фреймворк для построения REST API
- **Supabase** — база данных и хранилище файлов (PostgreSQL + storage)
- **Pydantic** — валидация данных и схемы
- **pytest** — тестирование
- **CI/CD** — настроен для автоматического тестирования

### Frontend
- **JavaScript / Vue 3** — основная библиотека для интерфейса
- **Vite** — сборщик и dev-сервер
- **Axios** — для запросов к API
- **Tailwind CSS / CSS** — стилизация компонентов
- **Vue Router** — маршрутизация страниц
- **Компоненты**:
  - GameCard, GameDialog, ReviewDialog
  - CoverDialog, ConfirmDialog
  - Layout, Header, Footer

## Установка и запуск

### Backend

1. Клонировать репозиторий:
```bash
git clone <URL_REPO>
cd backend
git checkout backend
```
2. Установить зависимости через Poetry:
```bash
poetry install
```

3. Создать файл .env на основе .env.example и указать ключи для Supabase.
4. Запустить сервер: 
```bash
poetry run uvicorn app.main:app --reload
```
API будет доступно по адресу `http://127.0.0.1:8000`
Документация Swagger: `http://127.0.0.1:8000/docs`
5. Запуск тестов:
```bash
poetry run pytest
```

### Frontend

1. Клонировать репозиторий:
```bash
git clone <URL_REPO>
cd frontend
git checkout frontend
```

2. Установить зависимости:
```bash
npm install
```

3. Создать файл .env и указать URL backend API.
4. Запуск dev-сервера:
```bash
npm run dev
```
5. Сборка для продакшена:
```bash
npm run build
```
